# Caricamento del Logo di MicDrop Studio

Caro utente, puoi caricare il tuo file logo PNG direttamente in questa cartella!

### Come Caricare il Logo:
1. Trascina (drag & drop) o carica il file immagine del tuo logo in questa cartella (`/public/assets/logo/`).
2. Assicurati che l'immagine si chiami esattamente **`logo.png`** (in minuscolo).
3. Una volta caricata, l'applicazione rileverà automaticamente la presenza del file all'indirizzo `/assets/logo/logo.png` e sostituirà il logo vettoriale automatico con la tua splendida immagine PNG!

Se non carichi alcun file, non preoccuparti: l'app userà la nostra replica vettoriale animata ad alta fedeltà che riproduce perfettamente i colori arancione e viola del brand!
