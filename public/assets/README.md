# MicDrop Studio Assets Directory

Place your logo and other custom media assets in this folder.
For the site's logo, please upload an image named `logo.png` directly into this folder.

### File Naming Convention
- **Logo**: `/public/assets/logo.png`
- **Fallback**: If `logo.png` is not uploaded, the application will automatically display an elegant, responsive CSS vector logo styled exactly like the original MicDrop orange-and-violet theme.
